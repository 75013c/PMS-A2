Help and Support

This page provides frequently asked questions and troubleshooting guidance for the Inventory Management System. It helps users understand common issues and how to solve them.

Frequently Asked Questions

How do I add a new item?

Open the Item Management page, choose Add Item, complete the form, and click Save Item.

How do I update an item?

On the Item Management page, choose Edit Item, enter the item name, click Load Info, make the changes, and then click Update Item.

How do I delete an item?

Choose Delete Item in Item Management, enter the item name, click Delete Item, and confirm the action in the confirmation box.

How do I search for an item?

Go to the Search Item page, enter the item name, and click Search to display matching results.

Troubleshooting Guidance

The item cannot be added

Check that all required fields have values. Make sure the Item ID is unique and the Item Name does not already exist in the system.

The search returns no result

Make sure the item name is spelled correctly. Try using a shorter keyword or check whether the item has been added to the inventory.

Changes are not saved permanently

This application uses browser session storage. Data may be cleared when the browser session ends, so the information is not stored permanently like a database system.

The layout looks crowded on small screens

Resize the browser window or use a larger screen width. The application supports responsive layout, but some sections may appear more compact on smaller devices.
